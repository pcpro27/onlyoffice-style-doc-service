# OnlyOffice-style Document Service

Ce projet permet de créer et éditer des documents en ligne, inspiré par OnlyOffice.

## Structure
- `frontend/` : Application React pour l'interface utilisateur.
- `backend/` : Serveur Express pour gérer les documents.
- `docs/` : Dossier pour stocker les documents (via GitHub API).

## Installation

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm start
```

## Intégration GitHub
Les documents sont sauvegardés via l'API GitHub (à configurer).
